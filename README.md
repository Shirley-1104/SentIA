# SentIA 💜

## Descripción

SentIA es una plataforma web de orientación vocacional inteligente diseñada para ayudar a los usuarios a descubrir posibles caminos profesionales según sus intereses, habilidades y características emocionales.

El sistema realiza un test interactivo con preguntas dinámicas y genera resultados personalizados mostrando las carreras con mayor compatibilidad, perfiles detectados y recomendaciones académicas.

---

## Características

✅ Test dinámico con preguntas condicionales  
✅ Resultados personalizados  
✅ Top 3 carreras con mayor compatibilidad  
✅ Perfil emocional detectado  
✅ Habilidades identificadas  
✅ Campo laboral asociado  
✅ Universidades recomendadas  
✅ Diseño responsive e interactivo
